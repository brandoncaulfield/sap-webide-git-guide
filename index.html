Important Concepts in SAP WebIDE

SAP WebIDE allows you to create a git repository for each project you work on. The creation of the repository is controlled by SAP WebIDE and if you want to work on the application you will need to clone the repository into your own git account.

Remebr, you have to commit and also push your changes to the remote repository if you want other developers to see your changes.

Create local repository in WebIDE


  1. In SCP create the remote Git repository (for the required URL)

  2. Then in WebIDE we set the remote repository for the app from the step above


  3. First pull from the newly created remote repository to merge your initial files with the remote master.


  4. Then push any changes you migh'tve made or just push anyway to be safe


  5. From there another user should be able to clone the newly created remote repository
